# Changelog

Semua perubahan penting pada project ini akan didokumentasikan di file ini.

## [2.0.0] - 2024 (Update Terbaru)

### ✨ Features Baru
- **Logo Custom di QR Code** 🆕
  - Upload gambar custom untuk ditampilkan di tengah QR code
  - Otomatis membuat background putih untuk logo
  - Clear logo dengan tombol dedicated
  - Support format: JPG, PNG, GIF
  
- **Warna Tema Baru** 🎨
  - Gradient tosca (#0F9B8C) ke putih (#F0FFFD)
  - Perpaduan warna aqua (#26D0CE)
  - Design lebih segar dan profesional
  - Better color contrast untuk accessibility

### 🎨 Improvements
- Update semua aksen warna dari purple ke tosca
- Better visual hierarchy dengan warna gradient tosca-aqua
- Logo preview dengan styling cantik
- Improved feedback messages

### 🐛 Bug Fixes
- Better handling untuk logo transparency
- Improved canvas rendering untuk QR code dengan logo

## [1.0.0] - Initial Release

### Features
- ✅ Generate QR code dari text/URL
- ✅ Customize warna foreground & background
- ✅ Adjust size (200px - 600px)
- ✅ Download PNG
- ✅ Copy to clipboard
- ✅ Reset settings
- ✅ Real-time preview
- ✅ Fully responsive design
- ✅ Beautiful UI dengan gradient background
- ✅ Toast notifications

### Technical
- Pure HTML/CSS/JavaScript (no frameworks)
- Single file deployment
- QRCode.js library for QR generation
- Responsive design mobile-first
- Cross-browser compatible
